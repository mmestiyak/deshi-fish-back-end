## Features

- User Can Sign in
  - Checkout
  - See Order List
- Admin can
  - add product
  - Manage Products
  - Manage Orders

- [Npm] - Package manager
- [Node JS] - javaScript Runtime Environment
- [Express js] - Back end framework
- [Mongo DB] - Database
